¿Quién soy yo?

Soy una persona apasionada por los dispositivos mecatrónicos, los sistemas electrónicos de este tipo de elementos utilizan programación Java o C, gracias a ello cuento con experiencia en desarrollo de páginas web (HTML)

Relacion Java y C con HTML y CSS(JavaScript)

Una página HTML gracias a su simple estructura permite una gran compatibilidad entre dispositivos, pero necesita de piezas clave para poder funcionar como un navegador web que puede estar desarrollado en lenguaje Java o C y de la necesidad de interactuar con el usuario nace JavaScript, el cual ayuda la adición de botones con movimientos especiales, bots de respuesta autónomos para páginas web interactivas y aplicaciones web interactivas.

JavaScript como complemento

Atraer la atención de un usuario para la venta de algún equipo resulta ser una tarea complicada, por eso un buen lenguaje de programación ineractivo es indispensable sobre una pantalla, ya que este te permitirá dar movimientos intuitivos y atractivos a tu cliente.

Conocimientos

Los conocimientos que adquirí a lo largo de mi vida me permiten diseñar modelos en 3D y 2D esos modelos pueden ser reales o surrealistas gracias a la amplia paquetería de diseño que nos brinda Adobe, Dassault Systemes, Autodesk, etc.

HTML5 & CSS3

HTML5 y CSS nos permiten realizar páginas web interactivas y con una apariencia intuitiva, gracias a el orden que mantienen entre ambos, con un .html diseñas el texto y ubicación de imágenes mientras que con .css le das una apariencia a el texto.

Ideas Creativas

Nuevos modelos de paginas web añadiendo JavaScript para dar tiempos de espera interactivos entre saltos de páginas HTML y con Photoshop y After Effects se establecen imágenes o videos interactivos en el fondo.

Facil De Editar

Se puede elaborar un sistema con Java el cual utilize una interface grafica que permita a un adminisrador hacer cambios sobre la página web de manera sencilla, sin necesidad de abrir el código.

Administrador(SQL &Angular)

Administrador para páginas web con ventas de productos tangibles o intangibles, agregar elementos basados en 'angular' el cual ayuda a generar bases de datos SQL, las cuales contienen nuevos productos, productos vendidos, cantidad de personas que ingresan a la página, esto proporciona administración en forma tablas y gráficos.
